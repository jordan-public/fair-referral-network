## To run:
- Clone this repository
- In the root folder type 
    ./setup.sh
    cd dapp
    npm install
    npm run dev
- The application pops up in the browser
- Use in conjunction with WorldCoin World ID Simulator any WalletConnect compatible wallet such as the Mobile MetaMask.
